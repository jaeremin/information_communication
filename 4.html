정보통신개론 chapter 8 연습문제

4. 네트워크 계층에서 동작하는 인터네트워킹 장비는?
1.브리지
2.라우터
3.리피터
4.서버

2번
브리지 : 물리 계층이 서로 다른 LAN 간에 연결(데이터 링크 계층) 라우터 : 동일 전송 계층 이상의 프로토콜을 사용하는 분리된 네트워크 연결, 네트워크층간 연결 리피터 : 동종 네트워크 범위 확장시 물리 계층 신호를 증폭하여 연결 허브 : 수신한 신호를 정확히 재생하여 다른 쪽으로 내보냄(물리 계층) 서버 : 인터넷(Internet)을 통해 클라이언트(Client)에게 서비스나 정보를 제공하는 역할을 수행하는 컴퓨터
